# 欢迎使用邮局服务

本指南帮助您快速上手邮局服务，涵盖从初始配置到程序对接的完整流程。

## 特色功能

- **网页控制面板**：通过直观的 Web 界面管理邮件服务。
- **多域名支持**：可添加并管理多个自有域名，满足不同业务场景。
- **隐藏源站 IP**：降低源站 IP 暴露风险，提升发信安全性。
- **多种对接方式**：支持 SMTP 与 API 对接，适配各类应用场景。
- **独享版本定制**：面向大用量用户提供独享部署，保障稳定性与性能，并支持多人协作使用。

## 使用须知

为确保邮件正常投递并维护域名信誉，请遵守以下规则：

- **关闭网站的每日邮件功能**。大量发送非必要邮件会被视为骚扰，导致域名信誉下降。
- **开启注册时的邮箱验证**，并启用人机验证，避免向无效或已失效的邮箱地址发信（如 `xxx@123.com` 等）。
- **禁止向未注册用户发送营销邮件**。推广、促销类邮件仅可发送给已明确同意接收的用户（Opt-in），不得向未注册或未订阅的用户群发。
- **邮件进入垃圾箱属于不可控因素**，没有任何服务商能保证 100% 进收件箱。建议使用 `.com`、`.net` 等国际顶级域名；`.top`、`.xyz` 等低价后缀更容易被标记为垃圾邮件。
- 为保证公平使用，共享产品设有 **3000 封/小时** 的发件速率限制。超出部分的邮件会被 HOLD，**且不会自动重发**（独享产品无此限制）。
- 邮局已**屏蔽所有临时邮箱域名**，发往临时邮箱的邮件将不会被投递。
- 邮局提供**验证码攻击检测**：当检测到大量标题含「验证码」的邮件时，系统会自动暂停发信。被封禁时，系统会发送邮件通知，并在邮件中说明自助解封流程。
