# 创建服务器

首次登录面板后，需要先创建邮件服务器。

![](/assets/images/build-server.png)

点击 **Build a new mail server**（创建服务器），在 **NAME** 处填写服务器名称（可随意填写），然后点击 **Build server** 完成创建。

![](/assets/images/create-server.png)

::: warning 注意
请勿创建多个 Server。单个 Server 即可绑定多个域名；若创建多个 Server，发件速率限制将按 Server 数量平摊到每个 Server 上，无法通过多开提高总限额。
:::
